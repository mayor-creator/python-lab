# 🐍 Python Lab

**Python Lab** is a hands-on learning repository for exploring Python through practical exercises, examples, and mini-projects. The goal is to build a strong foundation by learning through experimentation and real-world coding practice.

## 🎯 Objectives

* Learn Python fundamentals
* Practice problem-solving and programming logic
* Explore Python's built-in data structures and functions
* Write clean, readable, and maintainable code
* Build small projects to reinforce key concepts

## 📚 Topics

This repository may include lessons and examples covering:

* Python Basics
  * Variables and Data Types
  * Input and Output
  * Operators
* Control Flow
  * Conditional Statements
  * Loops
* Functions
* Modules and Packages
* Strings
* Lists, Tuples, Sets, and Dictionaries
* File Handling
* Exception Handling
* Object-Oriented Programming (OOP)
* Working with APIs
* Virtual Environments
* Unit Testing
* Python Best Practices

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/python-lab.git
```

2. Navigate to the project directory:

```bash
cd python-lab
```

3. (Optional) Create a virtual environment:

```bash
python -m venv .venv
```

Activate the environment:

**Windows**

```bash
.venv\Scripts\activate
```

**macOS/Linux**

```bash
source .venv/bin/activate
```

4. Run a Python script:

```bash
python lessons/example.py
```

## 💡 Learning Approach

Each lesson is designed to introduce a concept, followed by examples, exercises, and optional challenges. The emphasis is on learning by writing code, experimenting with solutions, and gradually building confidence.
